Python_Challenge includes two parts, pyBank and pyPoll. 

-pyBank: In this challenge, the tasked is about creating a script to analyze the financial records of the company.
	-What will we analyze in this python program:
		-The total Number of months included in this data
		-Net total amount of "Profit/Losses" over the entire period

		-The greatest increase in profits (date and amount) over the entire period
		-The greatest decrease in losses (date and amount) over the entire period

-pyPoll: In this challenge, the taksed is about creating a script to analyze a small, rural town modernize its vote counting process.
	-What will we analyze in this python program:
		-The total number of votes cast
		-A complete list of candidates who received votes
		-The percentage of votes each candidate won
		-The total number of votes each candidate won
		-The winner of the election based on popular vote.
